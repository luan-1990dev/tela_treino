# Guia do Usuário: Tela de Treino 🚀

O seu novo assistente pessoal para musculação e evolução constante.

## 📋 Como Organizar seus Treinos
*   **Abas A, B, C e D**: Organize sua divisão semanal de forma intuitiva.
*   **Adicionar Exercícios (+)**: Use o botão de "+" no topo para inserir novos movimentos na sua ficha.
*   **Reordenar**: Segure e arraste qualquer exercício para mudar a ordem deles.
*   **Excluir**: Use o ícone de lixeira vermelha para remover o que não faz mais parte da rotina.

## 📈 Evolução de Carga
*   **Registro por Série**: Cada série permite anotar o peso e as repetições de forma independente.
*   **Gráfico de Performance (📈)**: Clique no ícone de gráfico verde para ver uma linha do tempo da sua força. O app salva tudo automaticamente assim que você tica a série concluída!

## ⏱️ Cronômetro Inteligente
*   **Descanso Automático**: Ao marcar o checkbox de uma série, o timer de 60 segundos inicia sozinho.
*   **Ajustes Rápidos**: Use os botões de `+10s` ou `-10s` para ajustar o descanso.
*   **Toque para Parar**: O celular começou a vibrar? Basta **tocar em qualquer lugar da tela** para silenciar o alerta.
*   **Multitarefa**: Minimize o app e o cronômetro continuará visível em uma janela flutuante (PiP).

## 🌙 Personalização
*   **Temas**: Use o ícone de sol/lua no topo para alternar entre o Modo Claro e Modo Escuro para melhor conforto visual.
